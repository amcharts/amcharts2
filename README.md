# amcharts2

An official repository for amCharts JavaScript Charts V2 (free version).

**NOTE:** This is an old unsupported version. Use it at your own risk.

## Installing

### Using bower

```
bower install amcharts2
```

## Usage

Include `amcharts.js` on your web site:

```
<script src="/bower_components/amcharts2/amcharts/amcharts.js"></script>
```
